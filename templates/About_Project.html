Diabetic Retinopathy Detection: Project Overview

Introduction

Diabetic Retinopathy (DR) is a major cause of blindness, and early detection is crucial for effective treatment. This project classifies retina images into binary (No DR / DR) and multiclass (No DR, Mild NPDR, Moderate NPDR, Severe NPDR, PDR) categories using Machine Learning (ML), Deep Learning (DL), and Hybrid approaches.

Machine Learning Models (ML)

For ML-based classification, we extracted HOG (Histogram of Oriented Gradients) features from retina images and trained seven different models.

Preprocessing Steps for ML
  • Convert image to grayscale.
  • Resize image to 256x256.
  • Apply contrast enhancement and noise reduction.
  • Extract HOG features for model input.

Deep Learning Models (DL)

Deep Learning models use CNN-based architectures to extract high-level features automatically. We used MobileNetV2 and InceptionV3.

Preprocessing Steps for DL
  • Resize image to 299x299 for InceptionV3 and 224x224 for MobileNetV2.
  • Apply ImageNet normalization.
  • Extract features using pretrained models.
Models and Performance
Model Architecture Binary Accuracy Multiclass Accuracy
MobileNetV2 CNN 95.36% 75.85%
InceptionV3 CNN 96.32% 72.72%

Hybrid Models (DL + ML)

Hybrid models combine Deep Learning feature extraction (MobileNetV2/InceptionV3) with ML classifiers.

Preprocessing Steps for Hybrid
  • Extract features using MobileNetV2/InceptionV3.
  • Use ML models (SVM, MLP, RF, KNN, Logistic Regression, Naïve Bayes, Decision Tree) for classification.
